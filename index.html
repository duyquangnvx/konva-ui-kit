Konva UI Kit Demo - TypeScript

Bộ thư viện UI components với Konva.js và TypeScript

Button Component (TypeScript)

Nút bấm với hiệu ứng hover, click và hỗ trợ đầy đủ các sự kiện touch. Có thể tùy chỉnh màu nền, font chữ và bo góc.

const button = new Button({
  width: 150,
  height: 40,
  text: 'Click Me',
  backgroundColor: '#3498db',
  hoverColor: '#2980b9'
});
Label Component (TypeScript)

Component hiển thị văn bản với nhiều tùy chọn về căn chỉnh, phông chữ, màu nền và bo góc.

const label = new Label({
  width: 200,
  height: 40,
  text: 'Label Text',
  fontSize: 16,
  backgroundColor: '#f8f9fa',
  cornerRadius: 4,
  padding: 5
});
ImageView Component (TypeScript)

Hiển thị hình ảnh với nhiều chế độ hiển thị (contain, cover, fill, scale-down), hỗ trợ background, border và shadow.

const imageView = new ImageView({
  width: 200,
  height: 150,
  image: 'path/to/image.jpg',
  backgroundColor: '#f0f0f0',
  objectFit: 'contain',
  borderRadius: 6
});
